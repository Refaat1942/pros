Steps

أ — تثبيت

  1. Windows Update → Restart
  2. Laragon Full → Install → Start All
  3. Laragon → Menu → PostgreSQL → Install → Start All
  4. Git → Install
  5. تحقق: php -m | findstr pgsql → يظهر pdo_pgsql

ب — PostgreSQL (انسخ في psql)

Laragon → Menu → PostgreSQL → psql · أو CMD → psql -U postgres
CREATE DATABASE prosthetics ENCODING 'UTF8'; CREATE USER prosthetics_user WITH PASSWORD 'StrongPassword2026!'; GRANT ALL PRIVILEGES ON DATABASE prosthetics TO prosthetics_user; \c prosthetics GRANT ALL ON SCHEMA public TO prosthetics_user; ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON TABLES TO prosthetics_user; ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON SEQUENCES TO prosthetics_user;

ج — التطبيق

cd C:\laragon\www git clone https://github.com/Refaat1942/pros.git prosthetics cd prosthetics composer install --no-dev --optimize-autoloader copy .env.example .env php artisan key:generate

php artisan migrate --force php artisan storage:link php artisan config:cache php artisan route:cache php artisan view:cache

د — الشبكة والاختبار

  1. ipconfig → IPv4 → حدّث APP_URL → config:cache
  2. اختبار السيرفر: http://127.0.0.1
  3. اختبار PC آخر: http://SERVER-IP
  4. Firewall: Apache → Private ✓ · Public ✗

هـ — النسخ الاحتياطي

php artisan prosthetics:backup
  • الملفات: storage\backups · pg_dump · 7 أيام
  • Task Scheduler: يومياً 2:00 AM
  • نسخ أسبوعي لقرص خارجي

و — التسليم

  2. اختصار Desktop: http://SERVER-IP — «نظام الأطراف الصناعية»
  3. سلّم صفحة 3 لـ IT العميل
